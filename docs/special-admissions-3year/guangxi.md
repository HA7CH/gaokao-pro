# 广西 45 — 3 年特殊招生全景

**类型**: 自治区 | **改革**: 2023 旧高考 / **2024 起 3+1+2 首届**

## 高考改革节点

- **2023**: 旧文理(文/理)+ 旧艺体方案(综合分 文 40 + 艺 60)
- **2024**: 首届 3+1+2(物理/历史类) + 艺体新方案落地(50/50, 70/30)
- **2025**: 沿用 3+1+2;强基首达 105 计划
- **2026 → 130**(扩招 25 = +24%)

## 艺术统考(2024 新方案,6 类统一公式)

**综合分 = 高考总分 × 50% + 艺术统考 × (750/300) × 50%**(美术/音乐/舞蹈/表导演/播音/书法,全 50:50)

**2023 旧**: 综合 = 高考 × 40% + 艺术 × (750/300) × 60%(文化占比 40 → 50,新政硬约束)

**2025 合格线**(满分 300):

| 类别 | 合格线 |
|---|---|
| 美术与设计 | 200 |
| 音乐(表演) | 180 |
| 音乐(教育) | 170 |
| 舞蹈 | 190 |
| 表(导)演(戏剧影视表演) | 185 |
| 服装表演 | 195 |
| 戏剧影视导演 | 200 |
| 播音与主持 | 215 |
| 书法 | 225 |
| 戏曲 | 180 |

## 体育统招

- **公式**: 综合 = 高考总分 × 70% + 体育统考 × (750/100) × 30%(四舍五入取整)
- **本科合格线**: 体育统考 ≥ 83;文化线 253(23 旧)/309 历·300 物(24)
- **高职高专**: 体育统考 ≥ 60,文化 160/180

## 民族政策(重点)

### 民族加分梯度(满足"三统一" = 户籍 + 学籍 + 实际就读地三者均在桂同区域)

| 分值 | 适用范围 |
|---|---|
| **+15** | 10 世居少民(瑶/苗/侗/毛南/仫佬/回/彝/京/水/仡佬)在**南宁/柳州/桂林/梧州/北海 5 市城区以外** |
| **+15** | 融水、三江、龙胜 等 **28 自治县/享受自治县待遇县/山区县/边境县**少民 |
| **+15** | 防城区、东兴、靖西、那坡、凭祥、大新、宁明、龙州 **8 个边境县**考生 |
| **+7** | 融安、灌阳、蒙山 等 **22 山区县**少民 |
| **+5** | 上述之外的其他区内少民(满足三统一) |
| **+3** | 5 市城区少民 |
| **+7(封顶)** | 不满足三统一但中学户籍 + 高中学籍 + 就读地均在桂 |
| **+3** | 其他户籍在桂的少民 |

### 预科(重点)

- **预科 A 类(免费少数民族预科班)**: 仅面向广西 **60 个老少边山穷县(市、区)农村户籍少数民族应届考生**;录取按民族人口比例配额
- **预科 B 类(少数民族预科班)**: 面向**全区少数民族考生**(不限农村/应届)
- **投档**: 本批次最低提档线 **下 80 分内**;入校补一年高中,合格直升,不再高考

### 民族高校(区内主要承接)

广西大学、广西民族大学、广西师范大学、广西财经学院、玉林师范学院、广西中医药大学 等承接预科 A/B
