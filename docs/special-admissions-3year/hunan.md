# 湖南 43 — 3 年特殊招生全景

**类型**: 省 | **改革**: 3+1+2(2018 秋启动,**2021 首考**,第三批);本科批合并

## 艺术统考(2024 改革后,2024/2025 一致)

| 类别 | 公式 | 满分 |
|---|---|---|
| 音乐 | 文化 × 30% + 专业 × 70% | 750 |
| 舞蹈 | 同 30/70 | 750 |
| 表(导)演 | 同 30/70 | 750 |
| 美术与设计 | 同 30/70 | 750 |
| 书法 | 同 30/70 | 750 |
| **播音与主持** | **文化 + 专业(直接相加)** | **= 750 + 300 = 1050** |

**合格资格线**: **155**(2023/2024/2025 全科类全方向**统一,湖南"一刀切"**)

## 体育统招(湖南独特"直接相加")

- **投档分 = 高考文化(含政策加分) + 专业统考成绩**,**不加权、不折算**
- 文化满分 750 + 体育术科满分 300 = **理论上限 1050**
- 2025 本科首次投档: 文体须分别上控制线,按和值平行志愿排序

## 综评(在湘 8 校,2025)

中南大学(本省唯一本地校,**500 人 = 历 64 + 物 436**)、国科大、北外、南科大、深北莫、上科大、昆杜、上纽

**公式**: 高考成绩 + 面试考核成绩(等级制 A+~D:A+=19/A=18/A-=17/.../D=10);未涉学考会考

## 民族政策(湘籍 +10)

- **符合条件少民 +10**(同分排序时排序分不计加分;部分类型校投档可用)
- 需"三个三年": 户籍 3 年 + 学籍 3 年 + 实际就读 3 年
- **覆盖区**: 湘西土家苗自治州(吉首+凤凰+花垣+保靖+古丈+泸溪+永顺+龙山)、通道侗族、新晃侗族、靖州苗侗、芷江侗族、麻阳苗族、城步苗族、江华瑶族、永定区、武陵源区、桑植县
- 申报截止 2025/3/31(省民宗委审核)

## 关键发现

1. **"直接相加"是湖南独有口径**: 播音(1050) + 体育统招(1050)均不折百,与广东、四川等"7:3 折百"省份不同 — 评分卡需为湖南另开公式分支
2. **155 三年统一合格线**(2023/2024/2025 全类别)— 可硬编码
3. **3 校综评本地试点**: 中南大 + 国科大 + 北外 + 南科大 + 深北莫 + 上科大 + 昆杜 + 上纽 = 8 校,**中南大是唯一本省校(500 人)**
4. **国防科大 625 入围**: 湘籍考生量大但门槛与北方省份差距大(辽宁仅 620、河北 670)
5. **体育单招本省 3 招生校 + 1 文化考点**(湖南师大 3 个教学楼)
