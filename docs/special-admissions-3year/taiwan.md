# 台湾 71 — 3 年特殊招生全景

**类型**: 特殊地区 | **不在 31 省高考体系内,双向通道,方向 B 自 2020 历史性断裂**

## 双向通道

**方向 A 台籍 → 内地**(活跃且持续扩张)
- A1: 全国联招(港澳台华侨联招)
- A2: 凭学测成绩申请内地高校(2025/2026 已覆盖 **451 所**)
- A3: 福建对台单招(28 校统测 + 10 校分科测验 + 8 校单独招生 — 福建特有)
- A4: 暨大 + 华大两校联招(招台大户)
- A5: 港澳台居民居住证(2018+)— **不能单独凭此参加普通高考**,只解决就学/就业/社保

**方向 B 内地 → 台校**: **2020/04/09 大陆教育部全面暂停陆生赴台升读**,2024/25/26 仍未恢复

## 方向 B 内地 → 台校 政策变迁

| 年 | 事件 |
|---|---|
| 2011 | 台首次开放陆生学士班,2141 名额 |
| 2014-2016 | 本科 1700-2200/年峰值,在台陆生总数(含交换)~4w |
| 2017 | 蔡英文上任,本科名额压缩至 1000 |
| 2019 | 陆生硕博录取 1440 创新高,本科续降 |
| **2020/04/09** | **大陆教育部公告暂停陆生赴台升读**(COVID + 政治) — 历史性断裂点,仅允许在台应届陆生续读硕博 |
| 2021-2023 | 无新增学士班陆生 |
| 2024-2026 | **政策维持暂停**。台陆委会 2025/2026 多次呼吁通过小两会协商恢复,大陆未回应 |
| 现存 | 仅交换/短期研修(疫后部分恢复,规模极小);自费硕博零星个案 |

## 小三通 + 福建特色

- 小三通本身是厦金/福马客运通道(2025 厦金线 174w 人次),**与高考无直接挂钩**
- 但福建对台政策给台生**单独通道**:统测可报福建 28 校、分科测验可报厦大等 10 校、福建 8 校单独对台招生
- **福建是台籍考生入陆密度最高的省份**

## 重要政策时间线

- 2017: 蔡执政,陆生本科名额砍半
- 2018/09: 台胞居住证启动
- **2020/04/09**: 陆方全面停招赴台 — **方向 B 历史性断裂点**
- 2022/05: 福建 19 校试点台湾"统测"招生
- 2023/09: 中央国务院《福建探索两岸融合发展示范区》— 厦金/福马同城化、教育合作深化
- 2025/2026: 学测通道扩至 451 校,26 省 → 28 省覆盖

## 数据稳定性

**低-中**
- 方向 A 联招:台籍分项数据**仅 2024/2025 有公开拆分**,2023 及以前未官方按地区拆;录取人数完全无台籍单列
- 方向 A 学测:开放校数公开,**录取人数官方从未发布**
- 方向 B: **2020 后实际录取 ≈ 0**,但台方每年仍发简章(政策存续但通道关闭)
- 福建单招 / 两校联招数据**完全黑箱**
