## 体育统招

- 公式: 综合 = 文化 × 40% + 术科 × 2.5 × 60%
- 术科满分 300: 基础体能(100m + 立定三级跳 + 铅球,各 75) + 专项基础
- **2025 本科控制线**: 文化 365 / 术科 210
- **2023 本科控制线**: 文化 365 / 术科 203
