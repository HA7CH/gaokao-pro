# 香港 81 — 3 年特殊招生全景

**类型**: 特别行政区 | **不在 31 省高考体系内,走双向通道**

## 双向通道

**方向 A 港籍 → 内地高考**(3 路径)
- A1: 港澳台居民居住证(2018+)凭居住证 + 学籍参加普通高考(深圳/江苏/浙江/广东等省已纳入实操)
- A2: 全国联招(港澳台华侨联招),广州考点(暨大主办),4 科 750 分(中数英 + 文/理 3 科)
- A3: 港籍走 DSE → 内地承认院校

**方向 B 内地 → 港校**
- B1: 内地统招提前批(仅港中大 81002、港城大)
- B2: 港校独立招生(港大/港科技/港理工/港浸/岭南/教大/都大 等,不占内地志愿)

## DSE 互认现状(3 年扩张)

- 2023: ~138 内地高校承认 DSE
- 2025: **145** → 2025/11 报道 **165**(含大量双一流)
- 2024/25 学年: **4,562 名港籍生**通过 DSE 报读内地大学;热门:暨大 / 中山大 / 深大
- 普通考生最低标准: **3322A**(中/英≥3,数学≥2,公民科达标)

## 重要政策时间线

- 2018: 港澳台居民居住证制度
- 2023→2024: 联招报名 +42.58%(港籍激增)
- 2024/25: 八大非本地生 20% → 40% 阶梯开放
- 2025: 联招分数线暴涨 65~70 分
- 2025: 港都大门槛升至一本线
- 2026: 港校申请 +5.5%(8 年新高);港八大仅港中大保留统招提前批

## ⚠️ 区分(critical)

**港中大(深圳)/港科大(广州)/港大(深圳)**: 内地办学法人,**不应归入 81**,建议在数据中归 44 广东子集或单建 `mainland_branch_hk_universities` 表

## 数据稳定性

- 联招分数线 / 报名数:**高**(广东省考试院年度公布)
- 港中大、港城大提前批分数线:**高**(各省考试院交叉验证)
- 港大/港科大/港理工独立招生省别分数线:**中-低**(各校不公布,需第三方汇总)
- DSE 内地承认名单:**中**(年度更新,持续扩张)
- 2026 数据:**低**(招生季未完成)
