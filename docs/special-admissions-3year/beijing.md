# 北京 11 — 3 年特殊招生全景

**类型**: 直辖市 | **改革**: 3+3 (20 首届);23/24/25 沿用

## 艺术统考(2024 起 6 大类全市统考)

**公式**: 综合分 = 文化 × 50% + 专业 × (660/300) × 50% (≈ 文化 × 0.5 + 专业 × 1.1)

**合格线 (本科/专科)**:

**文化控线**: 23 非舞戏 336;24 326 / 舞戏 217;25 323 / 舞戏 215

## 体育统招

- **公式**: 北京"双过线后按高考总分排序投档"(平行志愿,无加权公式)
- **本科文化控线**: 23=390 / 24=400 / 25=389(均需术科 ≥60)
- **术科合格线**: 60(3 年统一)
- **术科 4 项**: 100m + 绕杆 + 立定跳远 + 原地推铅球(满分 100)

## 体育单招本省

- 文考考点: **北京市陈经纶中学(东大桥高中部)**
- 在京招生校: 北体大(全国 830 含保送)、首体院(运训 315 + 武民 80)、央财、北师大
- 国家统一公式: 文化/6 × 30% + 专项 × 70%
- 3 年扩招: 23 在 117 校 → 24 在 129 校 → 25 在 155 校(全国)

## 强基计划(在京 8 校 + 39 校面向京招)

- **在京 8 试点校**: 北大、清华、人大、北师大、中农、北航、北理工、央民
- **总名额 3 年**: 23 ~430 / 24 ~430 / 25 **473**(+18% 扩招)
- 清华在京 90 / 北大 87(3 年稳定)
- 25 清华入围 670(最低)、北大 87 名额维持

## 综评(在京 7 校外省 + 中外合办)

- 国科大 / 北外 / 上科大 / 南科大 / 上纽 / 昆杜 / 深北莫
- 公式: **631 模式**(高考 60% + 校测 30% + 学考 10%);北外 7:3
- 南科大 2025 在京录取综合分 84.87(物化)

## 民族政策

- 北京**无地方民族加分**(京籍少数民族无加分)
- 援疆援藏内地班单列,不属于本省特招通道
